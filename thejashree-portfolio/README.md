# Thejashree B – Portfolio

A single-page personal portfolio website for **Thejashree B**, showcasing data, AI, and analytics projects.  
Built with **HTML, CSS, and JavaScript**, inspired by the Sabo portfolio layout.

---

##  Features

- Hero section with name, role, and quick links.
- About section describing background and interests.
- Featured Projects with:
  - GitHub links to live code.
  - Custom SVG icons themed to each project.
- Contact section with:
  - Email
  - Phone
  - GitHub and LinkedIn links
- Smooth scroll navigation and scroll-reveal animations.
- Fully responsive layout for desktop and mobile.
